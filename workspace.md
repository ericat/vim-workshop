# Tasks and Examples
### Task 1: norm command
Delete inside tags:
```
<ul class="none">
  <li>@fa[thumbs-down] Delete is actually cut</li>
  <li>@fa[thumbs-down] Copy and paste can be tricky to setup</li>
  <li>@fa[thumbs-down] Global search and replace</li>
  <li>@fa[thumbs-down] Typos</li>
</ul>
```

```
:12,15norm dit
```
